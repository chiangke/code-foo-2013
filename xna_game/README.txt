INSTRUCTIONS
run xna_game/setup/setup.exe
the installer should automatically download required dependencies (Microsoft XNA 4.0 redistributable)
and start the game

To uninstall:
Uninstall/Remove Programs: GameStateManagement

CONTROLS:
WASD - move ship up/down/left/right
Left mouse button (press) - fire lasers
Left mouse button (hold) - charge shot 
Middle mouse button - fire bomb
Right mouse button - DO A BARREL ROLL
